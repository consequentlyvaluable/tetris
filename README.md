# Tetris

A fully responsive, touch-friendly Tetris built with HTML, CSS, and vanilla JavaScript.

## Play

Open `index.html` in your browser. The layout scales automatically for phones, tablets,
and desktops and includes on-screen controls for mobile players.

## Controls

- **Left / Right** arrows: move piece
- **Up** or **Z**: rotate
- **Down**: soft drop
- **Space**: hard drop
- **Touch buttons**: mirror the above controls on mobile

Use the Restart button to reset at any time. Level increases every 10 cleared lines and
speeds up the drop timer.
